# Extensions

Reusable collection, string, Unity object, and UI layout extension methods for Unity.

## Features

- Collection logging and random item selection
- String indentation and character filtering
- GameObject, Component, and Transform helpers
- Immediate and next-frame UI layout rebuilding

## Optional integrations

The `Highlight Plus Integration` sample adds highlighting and pointer-down binding helpers. Install Highlight Plus and Event Relays before importing the sample from Package Manager.

## Requirements

- Unity 2021.3 or newer
- Unity UI (`com.unity.ugui`)
